O que é o UnderLX?

O UnderLX é uma aplicação para dispositivos Android que mostra informação acerca do Metropolitano de Lisboa e recolhe dados anónimos sobre a sua utilização, para processamento e posterior apresentação de ainda mais informação. O objectivo é acompanhar o utilizador do Metro nas suas viagens, facilitando e tornando mais confortável a utilização deste meio de transporte

A informação disponibilizada inclui as perturbações e notícias conforme oficialmente publicadas, estatísticas sobre essas perturbações, informação sobre cada estação (tanto ao nível das suas características, como das correspondências com outros serviços de transporte), mapas de rede, etc

Os dados recolhidos servem para permitir o cálculo de várias estatísticas, como os tempos entre estações e os tempos de espera, a utilização relativa de cada estação, a quantidade de passageiros por hora do dia, etc. Estes dados serão gradualmente publicados assim que a quantidade de dados e os meios técnicos o permitam

Este é um projeto sem fins lucrativos e sem ligação oficial nem permissão explícita da empresa pública Metropolitano de Lisboa, E.P.E.. Tanto a aplicação como o software de servidor são de código aberto (Apache License 2.0)
