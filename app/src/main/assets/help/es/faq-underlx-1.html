¿Qué es el UnderLX?

UnderLX es una aplicación para Android que muestra información sobre el Metro de Lisboa y recoge datos anónimos sobre su uso, para análisis para que más información se pueda presentar más adelante. El objetivo es ayudar al usuario del Metro de Lisbon en sus viajes, facilitando y haciendo más cómodo el uso de este medio de transporte

La información disponible incluye noticias e interrupciones del servicio tal como se publicaron oficialmente, estadísticas sobre esas interrupciones (disturbios), información sobre cada estación, mapas de red, etc

Los datos recogidos permitirán calcular diferentes estadísticas, como el tiempo entre estaciones, el tiempo de espera medio, el uso relativo de cada estación, la cantidad de usuarios en función de la hora del día, etc. Estos datos serán analizados y publicados tan pronto como la cantidad de datos disponibles y los recursos técnicos lo permitan

Este es un proyecto sin fines de lucro sin una conexión oficial, respaldo o permiso explícito de la empresa pública Metropolitano de Lisboa, E.P.E.. Tanto esta aplicación como el servidor son de código abierto (Apache License 2.0)
